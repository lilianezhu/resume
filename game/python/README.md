# Space Shooter Python Game

## Run the game

From the project root, use the Python 3.11 virtual environment that was created for this project:

```powershell
cd C:\Users\arsli\OneDrive\Documents\resume
.venv311\Scripts\Activate.ps1
python game/python/game.py
```

If PowerShell blocks the activation script, run this once first:

```powershell
Set-ExecutionPolicy -Scope Process RemoteSigned
```

## Controls
- Move: WASD or arrow keys
- Shoot: Space
- Pause: P
- Restart: Enter or R

## Requirements
- Python 3.11
- pygame
- pytest
