# Space Shooter Game Proposal

## Game Concept
A classic 2D top-down space shooting game where the player pilots a starfighter through a scrolling starfield. The goal is to survive waves of enemy ships, collect power-ups, and defeat a final boss while avoiding incoming fire and obstacles.

## Core Idea
- Scene: outer space, with a moving starfield background.
- Player ship can move horizontally and vertically within screen bounds.
- Player can fire projectiles to destroy enemies.
- Enemies appear in patterns and attack the player.
- Score is earned by destroying enemies and collecting bonuses.

## Target Platform
- Desktop (Windows, Linux, macOS) using Python.
- Recommended library: `pygame` for graphics, audio, and input handling.

## Why This Game
- Fits the existing space shooting scene.
- Easy to prototype quickly in Python.
- Offers clear scope for fun gameplay and polish.
- Good foundation for later expansion, like multiple enemy types, levels, and score systems.

## Expected Features
- Player movement and shooting controls.
- Enemy waves with basic AI.
- Health or lives system.
- Score and HUD display.
- Simple background animation and sound effects.
- Pause and restart functionality.
